# Istruzione per arbitri rookie 
## Progetto di Carlo Bolis: programmatore python PRO
### Il mio progetto è un sito web creato per istrure i giovani arbitri di pallacanestro e aiutarli a svolgere più correttamente e professionalmente il loro lavoro. Il sito si sviluppa in 2 parti: 
- Sezione per l'istruzione: in questo paragrafo sono mostrate le principali infrazioni che possono essere commesse durante una partita ufficiale. 

- Sezione per metterdi alla prova: in questa sezione è presente un quiz per mettere alla prova gli arbitri e capire se sono pronti ad arbitrare correttamente o no!

### librerie da installare: 
 - Flask
 - Jinja2
